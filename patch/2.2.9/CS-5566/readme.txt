CS-5566: Event stills displays after delete
What is the problem to fix?
- Create new Event
- Find Event
- Select Event to delete
- Delete Event by Delete icon beside Show all Options
- Verify Event have been delete
Problem: Event stills displays.

Problem analysis
-The old normal behavior is that the delete button is not showed in search result. That mean in the search result, you can not delete calendar event.

How is the problem fixed?
- In search view: do not show delete button.
